features

Multi-agent council

Six roles — Caronte, Nettuno, Gerione, Plutone, Minosse, Lucifero — with feedback loops and promotion to skills.

Zelari-mode missions

Autonomous multi-run council until your MVP slice hits completion.ok or the iteration budget ends.

Rich TUI + tools

Ink + React chat, git sidebar, filesystem/shell/grep/web tools, 23+ skills, MCP via .zelari/ workspace.

Provider-agnostic

OpenAI-compatible APIs, xAI Grok OAuth, GLM/Z.AI — failover, headless CI, and /update self-update.
