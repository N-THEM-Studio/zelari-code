# Internal design notes (historical)

This directory holds **design docs, roadmaps, and implementation guides**
written during development. They are kept for archaeology and context.

**They may be outdated** relative to the current `main` branch and npm
release. For contributor-facing truth, prefer:

- [README.md](../../README.md)
- [docs/GUIDA.md](../GUIDA.md)
- [docs/TOOLS.md](../TOOLS.md)
- [CHANGELOG.md](../../CHANGELOG.md)
- [CONTRIBUTING.md](../../CONTRIBUTING.md)
- [docs/decisions/](../decisions/) (ADRs)

Do not treat filenames like `v0-6-0-roadmap.md` as the live product roadmap
unless the CHANGELOG still matches.
